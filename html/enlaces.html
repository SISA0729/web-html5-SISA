Enlaces

Los enlaces, o hipervínculos, son elementos cruciales en el diseño web, permitiendo la interconexión entre diversas páginas y recursos. La etiqueta principal para crear enlaces en HTML es <a>. Un enlace se define mediante la sintaxis.
Aquí, el atributo href especifica la dirección a la que el enlace conducirá al usuario. Es importante destacar que los enlaces no están limitados al texto; pueden ser aplicados a imágenes u otros elementos HTML.

HTML Enlaces - El Atributo Target

El atributo target dentro de la etiqueta <a> controla cómo se abre el documento vinculado. Los valores comunes para target incluyen _self (abre en la misma ventana/tab), _blank (abre en una nueva ventana o pestaña), _parent (abre en el marco principal), y _top (abre en toda la ventana).

URLs Absolutas vs. URLs Relativas

Los enlaces pueden utilizar URLs absolutas (direcciones web completas, como "https://www.ejemplo.com") o URLs relativas (ubicación dentro del mismo sitio web, como "/pagina.html").

HTML Enlaces - Enlazar a una Dirección de Correo Electrónico

Para crear un enlace que abra el programa de correo electrónico del usuario, se utiliza mailto: en el atributo href.

HTML Enlaces - Botón como Enlace

La creación de un enlace con un botón HTML implica el uso de JavaScript para definir las acciones a realizar en eventos específicos, como un clic en el botón.

HTML Enlaces - Títulos de Enlaces

La inclusión del atributo title proporciona información adicional sobre el enlace, mostrándose generalmente como un texto emergente cuando el mouse se sitúa sobre él. Este atributo es útil para ofrecer descripciones contextuales adicionales.
Atributo Target:

<a href="https://www.ejemplo.com" target="_blank">Abrir en nueva ventana</a>



URLs Absolutas vs. URLs Relativas:

<a href="https://www.ejemplo.com/pagina.html">Enlace absoluto</a>



<a href="/pagina.html">Enlace relativo</a>



Enlazar a una Dirección de Correo Electrónico:

<a href="mailto:info@ejemplo.com">Enviar correo electrónico</a>



Botón como Enlace (con JavaScript):

<button id="miBoton">Haz clic</button>
<script>
document.getElementById('miBoton').addEventListener('click', function() { window.location.href = 'https://www.ejemplo.com';
});
</script>



Títulos de Enlaces:

<a href="https://www.ejemplo.com" target="_blank" title="Visitar Ejemplo">Enlace con título</a>
