Estilos

Estilos y Colores:

CSS permite aplicar estilos y colores a elementos HTML, proporcionando una separación clara entre la estructura del contenido y su presentación visual.

Manipulación de Texto:

CSS ofrece control detallado sobre el texto, permitiendo ajustar propiedades como color, tamaño de fuente, espaciado entre letras y líneas (espaciado), alineación, decoración (subrayado, tachado) y transformaciones de texto (mayúsculas, minúsculas).

Cascada de Estilos:

La cascada en CSS significa que los estilos aplicados a un elemento padre se heredan por sus elementos hijos. Sin embargo, la especificidad y la importancia de los estilos pueden cambiar esta herencia. Este concepto facilita la aplicación consistente de estilos en todo un sitio web.

Uso de CSS

  • En línea (Inline): Utilizar estilos directamente dentro de elementos HTML usando el atributo style. Es útil para aplicar estilos específicos a elementos individuales.
  • Interno (Internal): Definir estilos dentro de la sección <style> en el encabezado (<head>) del documento HTML. Afecta a toda la página y es útil para páginas con un estilo único.
  • Externo (External): Enlazar a un archivo CSS externo mediante el elemento <link> en el encabezado. Permite la reutilización de estilos en múltiples páginas, facilitando la consistencia del diseño en todo el sitio.

Propiedades Comunes de CSS:

  • Color (color): No solo define el color del texto, sino que también se puede aplicar a fondos y bordes. Puede expresarse en términos de nombres de colores, códigos hexadecimales o valores RGB.
  • Fuente (font-family): Permite especificar el tipo de fuente para el texto. Puede ser un nombre de fuente genérico o específico, como "Arial", "Verdana", etc.
  • Tamaño de Texto (font-size): Controla el tamaño del texto. Puede especificarse en píxeles, ems, porcentajes u otras unidades.

Borde (border), Relleno (padding) y Margen (margin):

  • Borde (border): Define un borde alrededor de un elemento HTML. Se puede personalizar especificando el tipo de borde, el grosor y el color.
  • Relleno (padding): Define el espacio entre el contenido del elemento y su borde. Ayuda a mejorar la legibilidad y estética del contenido.
  • Margen (margin): Define el espacio fuera del borde del elemento. Controla la separación entre elementos adyacentes.

Enlace a CSS Externo:

Los archivos de estilo externo contienen reglas de estilo y se vinculan a las páginas HTML con el elemento <link>. Esto facilita la actualización global del diseño al modificar un solo archivo CSS.
